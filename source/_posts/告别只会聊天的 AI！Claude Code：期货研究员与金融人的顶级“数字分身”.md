以前你问 AI：“帮我总结这篇研报。”
现在你对 Claude Code 说：“去把这 10 个文件夹里的 50 份 PDF 全部读一遍，提取出关于‘天然橡胶’的产能数据，直接给我生成一个汇总表格。”
